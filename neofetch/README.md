### Install Tutorial

```bash
cd ~ # or any path you want

git clone https://github.com/Xuer04/Awesome-dotfiles.git ./dotfiles

mkdir -p ~/.config/neofetch

cp -r ./dotfiles/neofetch ~/.config/neofetch
```
